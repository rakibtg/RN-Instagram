# Demo: https://youtu.be/2G9mblKguLw

# Features:
- No frame drop
- Dynamic sticky filter header with Android support.
- Redux
- Dispatch based navigation
- Isolated components

# Setup
- Clone this repo
- Run `yarn` to install dependencies.
- Start simulator `yarn ios` or `yarn android`
